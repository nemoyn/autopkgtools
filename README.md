<h3 id="autopkgtools"><strong>autopkgtools</strong></h3>

<p>=============================================================================</p>

<p>English Description: <br>
Sometimes,we need to auto-package many apks for diff app stores or others, and then it just for you. <br>
This is a auto-package tools for auto change android apk’s  “Channel Code”, “App icon”, “package name”.</p>

<p>Chinese Description: <br>
自动打包工具，可以自动打包多渠道，多版本名称，多个应用名称的工具。</p>

<p>How to build <br>
1. install eclipse and fatjar plug-in <br>
2. Build Fat Jar <br>
3. Set Main-Class: tools.AutoTools</p>
